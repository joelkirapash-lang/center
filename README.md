# Weather Alerts App 🌦️

This project is part of the **External APIs and Fetching Lab**.

## 📌 Description
This web application allows users to enter a **US state abbreviation** (e.g. NY, CA) and fetch current weather alerts using the National Weather Service API.

## 🚀 Features
- Fetches weather alerts using an external API
- Displays number of alerts and headlines
- Handles errors for invalid input or failed requests
- Clears input after submission
- Dynamically updates the DOM

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

## 📡 API Used
- https://api.weather.gov/

## ▶️ How to Use
1. Open `index.html` in your browser
2. Enter a US state abbreviation (e.g. NY)
3. Click **Get Weather Alerts**
4. View results or error messages
